Rule34Everywhere

Rule34Everywhere is a Greasemonkey script for enriching your web browsing experience. Whenever you visit a website that references a tag in Paheal's Rule34 Database a small icon and link is added to directly to the page. On mouseover it will display a random image with that tag, and clicking will link you directly to its gallery.

Install Code

Greasemonkey (for Firefox) or Tampermonkey (for Chrome) must be installed to use Rule34Everywhere.

Rule #34: If it exists, there IS porn of it.

Have you ever been browsing the internet and thought to yourself, “Man, this boring page could really use some porn on it”? If so, look no further, Rule34Everywhere is the solution for you.

Install

Fast

The algorithm Rule34Everywhere uses is only O(n log m) where n is the number of words on the page and m is the number of tags in Paheal's database. As all computer scientists know, “if it ain't exponential, it ain't slow”, so you are sure to have a smooth experience browsing with Rule34Everywhere.

Private

Rule34Everywhere caches Paheal's tag database locally and only makes requests to Paheal when the dog image is hovered over. When images are fetched like this, no referrer is sent so Paheal cannot see your browsing. Note that when the link is clicked though, the referer is sent.

Discreet

Rule34Everywhere is inherently SFW until one of the indicator icons is hovered over. Noone is likely to notice the small dog icons showing up every once and a while on your browser, and even if they do, few could identify it as the favicon for the Rule34 database.

Advanced Settings New

Easily enable and disable Rule34Everywhere's auto-loading through the Greasemonkey context menu. When disabled, Rule34Everywhere can be manually run on a page, when the time is “just right.”
